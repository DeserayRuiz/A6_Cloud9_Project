Let's Get Started with Python and Django!

Execute these following steps to create you first Django and Python App!

  1. Sign up for a basic free Cloud 9 account
  2. Create a workspace. You can name the workspace whatever you'd like and give it whatever description you'd like. The important part while creating a workspace is to choose the Django template. Once you've filled out the appropriate fields and selected the Django Template, click the create workspace button at the bottom of the page.
  3. At the bottom of your new Cloud 9 workspace is a tab that says bash which is followed by so other text. Click into this and type the following... django-admin startproject project . You can name your project whatever you'd like. The name of your project will take the place of 'project' in the given command. You have now created the template of you Django and Python application.
